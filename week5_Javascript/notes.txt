---------------------------------- Javascript ----------------------------------
Client side programming language that enables interactive webpages by directly 
manipulating the Document Object Model (DOM) on a client's computer

Javascript is good at Event Driven Programming, which is programming based on 
events that happen on the web in which it responds to events by using event 
listeners and handlers 

A Javascript code is added into a HTML webpage code with <script> tags
Example:
    <script>
        // function (do something) 
    </script>

The "document.querySelector()" function looks for tags, ids and classes in a 
HTML webpage 
Examples:
    const h1 = document.querySelector("h1")             // Returns the first h1 tag
    const p = document.querySelectorAll("p")            // Returns all the p tags 
    const name1 = document.querySelector("#name")       // Returns the first name id 
    const name2 = document.querySelectorAll(".name")    // Returns all the name classes

The "document.addEventListener()" function listens for specific events 
Example:
    document.addEventListener("DOMContentLoaded", function() {
        // function (do something)
    });

Event Listeners can also be added onto Query Selectors
Examples:
    document.querySelector("form").onsubmit = function() {
        // function (do something)
    };
    document.querySelector("button").onclick = function() {
        // function (do something)
    };
    document.querySelector("select").onchange = function() {
        // function (do something)
    };

Note: onsubmit, onclick, onchange are Event Listeners added to Query Selectors 


------------------- Application Programming Interface (API) -------------------
Well defined structured way for services on the internet to communicate with each
other
Application Programming Interfaces (APIs) often transfer data in Javascript objects,
specifically in Javascript Object Notation (JSON)

Example JSON syntax:
   {
      "key": value,
      "key": value,
      "key": value
   }
